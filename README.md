# TWRP Samsung Galaxy A30

## How to compile it:

To build:
export ALLOW_MISSING_DEPENDENCIES=true

setup envirnoment
`. build/envsetup.sh`

then prepare
`lunch omni_a30dd-eng`

make the image
`mka recoveryimage`

# How to find the image built

`cd /out/target/product/a30`

see recovery.img
